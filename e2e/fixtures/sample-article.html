Machine Translation

Machine translation is a subfield of computational linguistics that investigates how to use software to translate text from one language to another.

Early approaches were based on dictionaries and grammar rules. Modern systems use large neural networks.

Neural machine translation achieved remarkable results and is now widely deployed in production systems.
